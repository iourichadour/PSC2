We apologize that you will need to change your installed PSChrome Firefox
installation source. Instructions below for removing and re-adding
PSChrome without losing settings.

Overview

  • The AMO (Addons.Mozilla.Org) team has an approval process for all Firefox Addons that takes weeks to be processed.
  • I'm now hosting the application on a secure server, replacing the AMO hosted application due to these long time delays.
  • Please use the procedures below to remove the old version and install the new version

Procedures

  • Open PSChrome Options and make a copy of your environment settings

    1. Go to the login page of and PeopleSoft page

    2. Click on the settings link under Environments

    3. Click the checkbox for "View Settings data"

    4. Copy the text data revealed when checking the box

    5. Paste the content into a new text document

  • Uninstall PSChrome

    1. Navigate to Tools -> Add-Ons

    2. Remove the old PSChrome Addon

    3. Close and Restart Firefox

  • Install New PSChrome

    1. Navigate to pschrome.com/chrome/PSChrome2/features.html

    2. Click the Firefox icon to install

  • Import environment settings

    1. Go to the login page of and PeopleSoft page

    2. Click on the settings link under Environments

    3. Paste the setting you earlier coppied out of the PSChrome settings tab

    4. Click the update button and then conform you want to replace the existing environment settings.

  • Enjoy PSChrome
