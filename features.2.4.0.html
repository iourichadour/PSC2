General Information

PSChrome offers some basic features for use by developers and users enabling a simplified user experience in PeopleSoft environments.

 

Environments List

A environment is automatically added when the PSChrome icon is clicked and the current tab is a PeopleSoft environment. Environments can be removed and renamed by clicking on the correct icon.

Version Information

Google Chrome: You can determine your PSChrome version by going to chrome://extensions


Firefox: You can determine your PSChrome version by going to Tools -> Add-ons (Ctrl-Shift-A) and then clicking the "Extensions" menu

 

 

Toolbar

The toolbar provides hightly valuable information and features on every PeoleSoft page.

  • Field Infromation

    To avoid having to look at each page in Application Designer the Record/Field information is provided on screen. PeopleSoft names its fields as RECORD_NAME so that information is displayed. When fields are shown as Radio, Selection and Checkbox there is more information provided such as translate values for Selection boxes.

  • Page Information

    PeopleSoft Componenet, Menu and Page information can be accessed by clicking the Page Information button. There can be additional information such as Aplication Server, Tools Version and User when available. Copy buttons are available for Menu, Component and Page since it is a common action.

    PeopleSoft allows for a user to click Ctrl-J to view page information. For Chrome it is pressing Ctrl-Shift-J twice.

  • Timeout Evasion

    Often a user has multiple windows open for long periods of time. If those windows are equiped with a refresh button then it will be used to keep the page refreshed before the normal timeout period. It is turned on/off for each environment and not a global setting for security reasons.

  • New Window

    Some environments have the new window button turned off. By clicking the new window icon you can open a new window or by doing a Ctrl-Click you can open just the component without any portal menu or headers.

  • Settings

    If the settings icon appears there is a setting applicable to that page. Some settings are global and some are environment specific. They will be labeled in the settings panel that appears so you will know the difference.

 

Process Monitor

The PeopleSoft Process Monitor is very click heavy. To complete actions you normally have to click more than two times. To quicken the interaction there are "Quick Links" added to the right of each process. These quick links allow you to do many actions with a single mouse click. PSChrome also automaticly will refresh your process monitor screen when there are active processes so you don't continue to click the refresh button.

 

Auto-Refresh

To remove the need of a users to repeatedly click the "refresh" button an auto-refresh functionality has been developed. When a user is on the Process Monitor component/page in PeopleSoft the refresh button will automaticly be clicked under the defined circumstance where the Processing Status meets the selected status(es). Auto-Refresh is enabled by default with a default time.

By default when a process is unposted or in the status of Bocked, Canceled, Initiated, Processing, Pending, Queued or Restart the page will be refreshed at a set but customizable time interval.

Quick Link

The Quick Links section provides 1 click options for many common task such as restarting a failed process, canceling a process, deleting a process, holding a queued process, repost unposted content, direct links to message logs and log files.

  • Restart: restart a process in No Success status
  • Hold: put a queued job in hold status
  • Cancel: cancel a queued or processing process
  • Delete: delete a non running process
  • View Message Logs: drill down to the message logs page<
  • View Log Files: drill down to the log files page
  • Repost Content: respost unposted content

 

 

Trace Log Analysis

The Trace Logs that are generated by PeopleSoft are very useful but complex and messing. By pasting the text of your trace log into the text field on the options page under "Log Analysis" you can extract the SQL commands from the log file contents.

After pasting the trace log into the text box click the "Analyze" button to generate the SQL.

My Favorites Enhancements

The "My Favorites" built in functionality in PeopleSoft is time saving although it make it frustrating sometime when you would like to know where the content reference actually resides. PSChrome will track your favorites everytime you open the "My Favorites" folder. When those favorited content references are present in your normal portal navigation browsing so when you navigate your the portal navigation they will have a different font color and size so they stand out.

The drill down icon (delete icon) will remove the "My Favorites" navigation folder and route you directly to the navigation folder for the selected content refrence.

General Settings

There are multiple general settings that can widly apply to each environment.

 

Timeout Evasion

Timeout Evasion works by using the existing "Refresh" button on PeopleSoft pages. Timeout evasion begins 60 seconds before the timeout warning message would appear. It gives you 30 seconds of warning with a countdown before the page refresh begins. Clicking the icon will enable or disable the feature for the current environment only.

  • Timeout Evasion ON
  • Timeout Evasion OFF

 

Correct History mode by default

The Correct History checkbox will be checked when available so that when you enter into a component it will automaticly be in correct history mode. This only applies on the search page and if you need to go into a different mode you can click the button at the bottom of the page.

 

Advanced Search by default

If the Advanced Search link is shown on the search page it will be clicked for you.
